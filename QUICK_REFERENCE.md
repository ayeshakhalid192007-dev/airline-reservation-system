```

```

| Option | Action | What You Need |
|--------|--------|---------------|
| **1** | View All Flights | Nothing - just browse |
| **2** | Search Flights | Origin city, Destination city |

## 🎫 Booking Checklist

Before you book, have ready:

- ✅ Flight number (e.g., PK-301)
- ✅ Passenger name
- ✅ Age
- ✅ Passport ID
- ✅ Contact number
- ✅ Email address
- ✅ Seat class choice (Economy/Business)
- ✅ Payment method (Credit Card/Cash/Points)
- ✅ Payment details

---

## 💳 Payment Information

### Credit Card:
- 16-digit card number
- Cardholder name
- Expiry date (MM/YY)
- CVV (3-4 digits)

### Cash:
- Amount to pay
- System calculates change

### Loyalty Points:
- Customer ID
- System deducts points

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't find booking | Check PNR format: PNR001 (not just 001) |
| Payment failed | Verify card number (16 digits), expiry (MM/YY), CVV (3-4 digits) |
| No seats available | Try different flight or seat class |
| Program won't start | Use run.sh or run.bat launcher |

---

## 💡 Important Tips

1. **Save your PNR number** - You need it to view/cancel bookings
2. **Double-check details** - Verify passenger info before confirming
3. **Choose seat class carefully** - Economy is cheaper, Business is premium
4. **Have payment ready** - Speeds up the booking process
